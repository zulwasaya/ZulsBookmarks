This archive contains a snapshot release of MacVim 7.3. It requires Mac OS X 10.8 (Mountain Lion) or later. It will NOT run on earlier versions of Mac OS X! To install, drag the MacVim icon into the Applications folder. For usage instructions, please start MacVim and type ":h macvim" (without the quotes). This archive also contains the script "mvim" which can be used to start MacVim from Terminal. Type ":h mvim" in MacVim for help on how to install it.
